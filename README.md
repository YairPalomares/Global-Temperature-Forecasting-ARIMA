# Global-Temperature-Forecasting-ARIMA
Time series analysis of annual global temperatures (1870-2020) using ARIMA modeling. Applied Dickey-Fuller tests for stationarity and optimized performance via AIC across multiple configurations. The final ARIMA(1,1,1) model achieved a high accuracy with a 1.5% MAPE and an RMSE of 0.98, effectively capturing historical climate trends.
